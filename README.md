# TechCase

TechCase는 기업 기술 블로그를 기반으로, 개발자가 새로운 기술 스택 도입이나 기술적 이슈 해결을 고민할 때 실제 기업 적용 사례를 빠르게 탐색할 수 있도록 돕는 기술 사례 검색 서비스입니다.

기존 기술 블로그 모음 서비스는 여러 글을 한곳에서 볼 수 있다는 장점이 있지만, 검색 품질이 단순 키워드 매칭에 머무르는 경우가 많습니다. 예를 들어 Kafka, Kubernetes, Terraform, Redis, Elasticsearch 같은 기술을 검색하더라도 해당 키워드가 포함된 글만 나열될 뿐, 그 기술이 어떤 문제를 해결하기 위해 도입되었는지, 어떤 아키텍처 안에서 사용되었는지, 실제 서비스 맥락에서 어떤 의미를 가졌는지 파악하기 어렵습니다.

TechCase는 단순한 기술 블로그 모음이 아니라, 신뢰도 높은 기업/공식 기술 블로그를 기반으로 한 기술 사례 탐색 서비스를 목표로 합니다.

## 문제 정의

개발자가 새로운 기술 스택 도입을 검토하거나 운영 중인 시스템의 이슈를 해결하려고 할 때 공식 문서, 튜토리얼, 개념 설명은 비교적 쉽게 찾을 수 있습니다. 하지만 실제 기업들이 비슷한 문제를 어떤 배경에서 겪었고, 어떤 기술을 선택했으며, 어떤 구조로 해결했는지 정리된 자료를 찾기는 어렵습니다.

일반 검색에서는 다음과 같은 문제가 자주 발생합니다.

- 광고성 콘텐츠가 검색 결과에 섞입니다.
- SEO 중심의 얕은 글이 많습니다.
- 단순 개념 설명이나 설치 튜토리얼이 많이 노출됩니다.
- 실제 기업 적용 사례를 찾는 데 시간이 오래 걸립니다.
- 글 제목만 보고는 실무 맥락을 파악하기 어렵습니다.
- 기술명, 문제 상황, 아키텍처 키워드를 함께 검색하기 어렵습니다.
- 특정 이슈를 해결한 사례와 기술 도입 배경을 연결해서 보기 어렵습니다.

TechCase는 신뢰할 수 있는 기업 개발 블로그와 공식 기술 블로그를 우선 수집하여, 개발자가 실제 사례 중심으로 기술을 탐색할 수 있게 만드는 것을 목표로 합니다.

## 핵심 아이디어

TechCase는 단순히 기술 블로그 글을 모으는 서비스가 아닙니다.

TechCase는 기술이 실제 서비스에서 어떤 문제를 해결하기 위해 사용되었는지 탐색하는 서비스입니다.

예를 들어 사용자가 `Kafka`를 검색하면 단순히 Kafka라는 단어가 포함된 글만 보여주는 것이 아니라, 다음과 같은 맥락의 사례를 함께 찾을 수 있어야 합니다.

- 이벤트 기반 아키텍처
- 로그 수집 파이프라인
- 실시간 데이터 처리
- 서비스 간 비동기 통신
- CDC 기반 데이터 동기화
- Kafka와 Spring Boot, Kubernetes, 클라우드 서비스가 함께 사용된 사례

핵심 가치는 개발자가 기술 스택을 도입하거나 문제 해결 방향을 정하기 전에, 실제 기업에서는 어떤 문제 때문에 해당 기술을 선택했고 어떤 방식으로 적용했는지 이해할 수 있게 하는 것입니다.

## 주요 검색 축

TechCase는 하나의 검색창에서 단순 키워드 검색만 제공하는 것이 아니라, 여러 관점으로 기술 사례를 탐색할 수 있어야 합니다.

초기 검색 축은 다음과 같습니다.

- 기술명 검색: `Kafka`, `Redis`, `Kubernetes`, `Terraform`
- 클라우드 서비스 검색: `Lambda`, `EKS`, `DynamoDB`, `Kinesis`, `EventBridge`
- 회사/출처 검색: `AWS`, 이후 `Toss`, `Kakao`, `Naver`, `LINE`
- 문제 상황 검색: `cost optimization`, `high availability`, `large-scale traffic`, `migration`
- 아키텍처 검색: `event-driven`, `serverless`, `microservices`, `CDC`, `GitOps`
- 기술 조합 검색: `Kafka + CDC`, `Lambda + EventBridge`, `EKS + GitOps`

## 이후 확장 아이디어

- 한글/영문 기술명 동의어 검색
- `k8s`와 `Kubernetes` 같은 약어/별칭 검색
- 오타 보정 및 부분 검색
- 글에서 사용 기술 자동 추출
- 문제 상황 및 해결 방식 분류
- 글 기반 사례 요약
- 기술별 대표 사례 페이지
- 관련 기술 및 유사 사례 추천
- 문제 상황 기반 벡터 검색

## 한 줄 정의

TechCase는 신뢰도 높은 기업 기술 블로그를 기반으로, 개발자가 기술 스택 도입이나 기술적 이슈 해결을 고민할 때 실제 기업 적용 사례를 빠르게 탐색할 수 있도록 돕는 기술 사례 검색 서비스입니다.

## 상표 안내

TechCase에 표시되는 회사명과 로고는 각 소유자의 상표입니다. TechCase는 검색 결과의 출처 식별을 위해 로고를 사용하며, 각 회사와의 공식 제휴 관계를 의미하지 않습니다.
